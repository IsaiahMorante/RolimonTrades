# RolimonTrades

This is a barebones text representation of the recent trades ad page by Rolimon. I am using the api's they provided and organizing it in a readable format. <br />
This is more of a test to mess around with API's and to learn. To change any features, go into the python file and look for the configuration section. <br />

COMMANDS <br />
python RolimonTrades.py <br />
python RolimonTrades.py --clear-cache <br />
python RolimonTrades.py --amount <br />
